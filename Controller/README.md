How to run the app | Qt Creator
- add Custom Executable
- set python path "...\Controller\.qtcreator\Python_3_13_3venv\Scripts\python.exe"
- set command line args "./src/main.py"
- set working dir "...\Controller"

How to run the app | Terminal
- run $env:PYTHONPATH = "src" for powershell
- run "python .\src\main.py" in root dir

How to run unit tests | Terminal
- run "pytest" in root dir

How to install dependecies | Terminal
- run "pip install ." in root dir
